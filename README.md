# QRReader
A QR Code Scanning Framework For IOS 

## Requirements:
- IOS 11+
- Xcode: 12+

## Installation:
- steps to install this framework in your xcode project

- Pod:
- pod init

- Podfile:
```ruby
  platform :ios, '10.0'
  pod 'QRReader'
```
- then run pod install
